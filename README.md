# Currency-Converter

# 💱 Currency Converter Web App

A simple and responsive **Currency Converter** built using **HTML, CSS, and JavaScript**.
This project uses **Fetch API** to get real-time exchange rates and allows users to convert currencies instantly.

## 🚀 Features

* 🌍 Supports **150+ international currencies**
* 🔄 **Real-time currency conversion**
* 🚩 Displays **country flags** for selected currencies
* 🔁 **Swap currency button** for quick conversion
* 📱 **Responsive design** for mobile and desktop
* ⚡ **Fast conversion using Fetch API**

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Exchange Rate API
* Flags API
